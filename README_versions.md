Resumo de versões

- `simulacao_materia.py` — arquivo original do projeto (atual, contém mudanças recentes).
- `simulacao_materia_v1_original.py` — Versão 1: cópia da simulação original antes de integrar o Algoritmo Genético.
- `simulacao_materia_v2_genetic.py` — Versão 2: cópia da versão atual com o Algoritmo Genético integrado; contém `run_ga_example()`.

Como executar

- Rodar a simulação original:

```powershell
python simulacao_materia.py
```

- Rodar a versão com GA (exemplo):

```powershell
python -i simulacao_materia_v2_genetic.py
# no prompt do Python:
>>> run_ga_example()
```

Notas

- A versão v2 tem uma simulação reduzida para avaliação rápida do GA.
- Se quiser que eu ajuste nomes ou crie um histórico com mais versões (v3, v4, etc.), diga quais mudanças quer versionar.
